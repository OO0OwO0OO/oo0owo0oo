Project 3: Personal Portfolio Website

This personal portfolio website itself is a project that showcases my skills in web development. Built with Jekyll, HTML, CSS, and JavaScript, it demonstrates responsive design principles and modern web development practices.

Technologies and Sources Used:

  • Jekyll: Static site generator for building the website structure
  • HTML5: Semantic markup for content structure
  • CSS3: Modern styling with Grid and Flexbox for responsive design
  • JavaScript: Interactive features and mobile navigation
  • Font Awesome: Icon library for social media and UI icons
  • Git & GitHub: Version control and hosting via GitHub Pages

The website features a clean, modular structure, responsive navigation, and is fully optimized for various screen sizes. It represents my understanding of front-end development, version control, and deployment practices.
